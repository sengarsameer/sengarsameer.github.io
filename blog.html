Editors in Linux :-

Majorly you listen or know about these two editors Nano and Vim. Nano is the easiest to use and learn. A lot of people will swear by Vim but Nano is a very good text editor. Nano is good editing a config file but if you are going to program you'll be better off using Vim. Nano Supports highlighting. But this is very subjective.

Everyone is going to have their favourite editors for some reason. Find the one that you like the best and use it.
